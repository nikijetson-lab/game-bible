# 🗺️ Мапа Світу Хейзмур

## Загальна Схема

```
                    Північні Пустки
                         |
                   [Валькорн]
                  /     |     \
                 /      |      \
        [Грейфорд]----[Болота]----[Тихий Шелест]
                 \      |      /
                  \     |     /
                   [Затоплена Обитель]
                         |
                    Південні Трясовини
```

## Легенда (ASCII-мапа)

```text
    ⛰️⛰️⛰️⛰️⛰️⛰️⛰️⛰️⛰️⛰️
    ⛰️        🏰        ⛰️     ← Валькорн (столиця)
    ⛰️       /  \       ⛰️
    ⛰️   🏘️----🌿️----🏚️    ⛰️  ← Грейф—Болота—Тихий Шелест
    ⛰️       \  /       ⛰️
    ⛰️        💀        ⛰️     ← Затоплена Обитель
    ⛰️⛰️⛰️⛰️⛰️⛰️⛰️⛰️⛰️⛰️
         🚤 [Сонк-Феррі]
```

## Детальна Мапа (текстова)

### Зона 1: Валькорн 🏰
```text
[Вхід] → [Торгова площа] → [Замок]
                     ↘ [Трущоби]
                     ↘ [Церква Моура]
                     ↘ [Колодязь Бажань]
```

### Зона 2: Болота Хейзмуру 🌿
```text
[Вхід] → [Болотяна стежка] → [Центр болота]
                        ↘ [Печера]
                        ↘ [Стародавнє дерево]
                        ↘ [Схрон контрабандистів]
```

### Зона 3: Тихий Шелест 🏚️
```text
[Міст] → [Селище] → [Дім Старійшини]
               ↘ [Таверна]
               ↘ [Маяк (зруйнований)]
               ↘ [Кладовище]
```

### Зона 4: Грейфорд 🏘️
```text
[Ворота] → [Ринок] → [Адміністрація]
                    ↘ [Казарми]
                    ↘ [Портові склади]
```

## Кольорова Схема

```text
Зони:
  🟢 Безпечна зона (місто)
  🟡 Помірна небезпека (дороги)
  🟠 Небезпечна зона (болото)
  🔴 Смертельно небезпечно (обитель)
  ⚪ Особлива зона (квестові)

Позначки:
  🏰 Столиця
  🏘️ Місто
  🏚️ Селище
  ⛪ Святилище
  💀 Небезпечне місце
  🚤 Транспорт
  ❓ Таємне місце
```

## Розміри та Відстані

```text
Валькорн → Грейфорд:    1 день верхи
Валькорн → Болота:      2 дні верхи
Грейфорд → Тихий Шелест: 3 години пішки
Тихий Шелест → Обитель:  1 година пішки
Грейфорд → Сонк-Феррі:  30 хвилин пішки
```

## Рівні Висот

```text
Валькорн:         250м над рівнем болота
Грейфорд:         100м над рівнем болота
Тихий Шелест:     20м над рівнем болота
Болота:           0м (рівень води)
Затоплена Обитель: -15м (під водою)
Південні Трясовини: -5м (на 50% під водою)
```

## Ключові Маршрути

```text
1. Валькорн → Грейфорд (Головний тракт)
   Час: 1 день
   Небезпека: 🟡 Низька (розбійники рідко)
   
2. Валькорн → Болота (Старий лісовий шлях)
   Час: 2 дні
   Небезпека: 🟠 Середня (вовки, болотяні)
   
3. Грейфорд → Тихий Шелест (Болотяна стежка)
   Час: 3 години
   Небезпека: 🟠 Середня (трясовина)

4. Тихий Шелест → Затоплена Обитель (Таємний прохід)
   Час: 1 година
   Небезпека: 🔴 Висока (Муровані створіння)

5. Сонк-Феррі → Всі напрямки (Болотяний пором)
   Час: 1-4 години
   Ціна: 5 золотих
```
