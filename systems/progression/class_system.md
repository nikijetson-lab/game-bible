# 🎭 Класова Система

## Архетипи Персонажа

### 1. Вартовий (Warrior) — Базовий клас
```text
Спеціалізація: Ближній бій
Стартові стати: STR 7, DEX 5, INT 3, END 6, WIS 4
HP: 70 | MP: 15

Унікальні здібності:
- Стійкість: +20% до всієї шкоди
- Розмахування: атака по площі (3 вороги)
- Бойовий крик: +20% шкоди на 10 сек

Доступна зброя: мечі, сокири, булави
```

### 2. Тінь (Shadow) — Стелс/вбивця
```text
Спеціалізація: Стелс, критичні удари
Стартові стати: STR 4, DEX 8, INT 4, END 4, WIS 5
HP: 45 | MP: 25

Унікальні здібності:
- Тіньова хода: +50% швидкості в стелсі
- Удар зі спини: x3 шкоди (потрібен стелс)
- Відмичка: +30% до зламування замків

Доступна зброя: кинджали, луки, отрути
```

### 3. Культист (Cultist) — Магія/прокляття
```text
Спеціалізація: Магія крові та тіней
Стартові стати: STR 3, DEX 4, INT 8, END 3, WIS 8
HP: 35 | MP: 50

Унікальні здібності:
- Кривавий договір: +20% шкоди, -10% HP
- Прикликання: викликати тіньового слугу
- Жертвоприношення: конвертувати HP в MP (1:2)

Доступна зброя: посохи, кинджали, сувої
```

### 4. Мисливець (Hunter) — Дальній бій/виживання
```text
Спеціалізація: Дальня атака, пастки
Стартові стати: STR 5, DEX 7, INT 4, END 5, WIS 4
HP: 55 | MP: 20

Унікальні здібності:
- Влучне око: +30% шкоди з лука
- Пастки: встановити ведмежу пастку (50 шкоди)
- Слідопит: бачити сліди ворогів на міні-карті

Доступна зброя: луки, арбалети, списи
```

### 5. Проклятий (Cursed One) — Спеціальний клас
```text
Відкривається після квесту "Кров для Болотяного Бога"
Спеціалізація: Жива зброя, спотворення

Стартові стати: STR 6, DEX 5, INT 5, END 5, WIS 5
HP: 60 | MP: 35

Унікальні здібності:
- Кривавий голод: живий меч вимагає жертв
- Спотворення: +50% шкоди, але -30% HP
- Око Моура: бачити невидиме

Доступна зброя: тільки жива зброя
```

## Система Прокачки Класу

```text
Кожен рівень:
- +10 HP (база)
- +5 MP (база)
- +3 очки статів

Кожні 5 рівнів:
- Нова унікальна здібність класу

На 10, 20, 30, 40, 50 рівні:
- Вибір перку (посилення існуючої здібності)
```

## Зміна Класу

```text
Можна змінити клас тільки через особливі події:
1. Вівтар Моура → стати Проклятим
2. Храм Валькорну → стати Вартовим
3. Тіньовий Ринок → стати Тінню
4. Зустріч з культистами → стати Культистом

Зміна класу: 
- Скидає 30% очок статів
- Зберігає 50% вивчених здібностей
- Відкриває нове дерево навичок
```
