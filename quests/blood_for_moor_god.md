# 🩸 Кров для Болотяного Бога

## Квест: "Blood for the Moor God"

### Ініціація
**Умова**: Меч досяг 3 рівня (10 жертв)
**Місце**: Затоплена Обитель, вівтар під водою
**NPC**: Голос Моура (безтілесний, чути всюди в болотах)

### Діалог Ініціації
```
Голос Моура: Ти чув поклик плоті... ти чув поклик крові...
Голос Моура: Прийди до мене. Принеси три жертви на моєму вівтарі.
Голос Моура: Я дам тобі силу. Справжню силу.

[Прийняти] → Quest Started
[Відмовити] → "Ти ще повернешся. Вони всі повертаються..."
```

### Етапи Квесту

#### Етап 1: Жертвоприношення
```gdscript
func stage_one():
    objectives = [
        "Знайти вівтар Моура в Затопленій Обителі",
        "Принести в жертву розбійника (0/1)",
        "Принести в жертву стражника (0/1)",
        "Принести в жертву невинного (0/1)"
    ]
```

- **Розбійник**: Легко, скрізь на болотах
- **Стражник**: Складніше, патрулюють Валькорн
- **Невинний**: Найважче, моральний вибір (селянин, дитина, чернець)

#### Етап 2: Ритуал Спотворення
```gdscript
func stage_two():
    objectives = [
        "Занурити меч у кров на вівтарі",
        "Вимовити прокляття давньою мовою",
        "Пережити зустріч з Аватаром Моура"
    ]
    
    # Ефекти після ритуалу
    GameManager.add_perk("blessing_of_moor")
    # +30% шкоди, -50 репутації з усіма фракціями
```

#### Етап 3: Фінальне Випробування
```gdscript
func stage_three():
    objectives = [
        "Убити 5 послідовників Моура (відступників)",
        "Повернутися до вівтаря з їхньою кров'ю",
        "Занурити меч востаннє"
    ]
    
    # Фінальні нагороди
    rewards = {
        "sword_upgrade": "Рівень 5 (Бог Війни)",
        "perk": "moor_favor",  # Невразливість у болотах
        "curse": "eternal_hunger"  # Меч більше ніколи не насититься
    }
```

### Наслідки

#### Для Світу
- **Репутація**: -100 з усіма фракціями
- **NPC**: Бояться гравця, деякі тікають при появі
- **Локації**: Нові культові табори з'являються

#### Для Гравця
```gdscript
func apply_consequences():
    # Фізичні зміни
    player.skin_color = Color(0.3, 0.3, 0.3)  # Сірий відтінок
    player.eyes_glow = true  # Очі світяться червоним
    player.passive_aura = "fear_aura"  # Аура страху
    
    # Ігрові зміни
    player.speed *= 1.2  # Швидкість +20%
    player.damage *= 1.5  # Шкода +50%
    player.max_hp *= 0.7  # HP -30% (меч пожирає плоть)
    
    # Додаткові ефекти
    add_periodic_event("bog_spirits", 300)  # Духи боліт атакують щохвилини
```

### Альтернативна Кінцівка
**Якщо гравець відмовляється вбивати невинного:**

```gdscript
func alternate_ending():

```
