# ⚖️ Баланс та Числа

## Характеристики Персонажа

### Базові Стати
```text
Сила (STR):     впливає на шкоду ближнього бою   (база: 5)
Спритність (DEX): впливає на ухилення та швидкість (база: 5)
Інтелект (INT):   впливає на магію              (база: 5)
Витривалість (END): впливає на HP               (база: 5)
Мудрість (WIS):    впливає на репутацію          (база: 5)

HP = 50 + END * 10
MP = 20 + INT * 5
Швидкість = 100 + DEX * 5
Шкода (меч) = 5 + STR * 2
Шкода (магія) = 5 + INT * 3
```

### Рівні та Прокачка
```text
XP для рівня = поточний_рівень * 100 + 50

Рівень 1:  0 XP  → HP: 50,  Стати: 5 кожен
Рівень 2:  150 XP → HP: 60,  +3 очки статів
Рівень 5:  750 XP → HP: 100, +12 очок статів
Рівень 10: 2550 XP → HP: 200, +27 очок статів
Рівень 20: 10550 XP → HP: 400, +57 очок статів
Макс рівень: 50
```

```text
Іржавий меч:        шкода 8-12,  швидкість 1.0
Сталевий меч:       шкода 12-18, швидкість 1.0

Живий меч (рів 1):   шкода 10-15, швидкість 1.0 (+кровожадливість)
Живий меч (рів 5):   шкода 30-45, швидкість 1.2 (+кров, +отрута, +страх)
```

### Дальній Бій
```text

Арбалет:            шкода 10-15, дальність 150
Магічний посох:     шкода 8-15,  дальність 200
```

### Магія
```text
Школа Крові:
- Удар кров'ю:     шкода 10-15, ціна 20 MP
- Кривавий дощ:    шкода 5/сек по площі, ціна 50 MP
- Жертва:          +50 HP, -30 MP

Школа Тіней:
- Тіньовий кинджал: шкода 15-20, ціна 30 MP
- Невидимість:      30 сек, ціна 40 MP
- Темрява:          осліплює ворогів на 10 сек

Школа Болота:
- Отруйний туман:   шкода 8/сек, ціна 25 MP
- Прикликання щупалець: шкода 15, ціна 35 MP
```

## Формули Розрахунку

### Шкода
```text
Фінальна шкода = (базова_шкода + STR*2) * модифікатор_зброї * (1 + бонуси_за_рівень*0.1)
                 - захист_цілі / 2

Шанс критичного удару = DEX * 0.5%
Критичний множник = 2.0 + STR * 0.05

Шанс ухилення = DEX * 1.0% (макс 50%)
```

### Ціни в Магазинах
```text
Базова ціна * модифікатор_репутації

Репутація 100:  x0.70 (знижка 30%)
Репутація 50:   x0.85 (знижка 15%)
Репутація 0:    x1.00 (базова)
Репутація -50:  x1.25 (націнка 25%)
Репутація -100: x1.50 (націнка 50%)
```

### Досвід за Вбивство
```text
XP = HP_ворога * 0.5 + шкода_ворога * 1.0 + бонус_за_тип

Звичайний:  x1.0
Елітний:    x2.0
Бос:        x5.0
```
