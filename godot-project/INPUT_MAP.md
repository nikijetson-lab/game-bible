# Input Mapping Configuration for Hazemoor
# Налаштування керування

## Щоб додати в Godot:
## Project → Project Settings → Input Map

## Movement
move_forward: W, Up Arrow
move_back: S, Down Arrow
move_left: A, Left Arrow
move_right: D, Right Arrow

sprint: Shift (Left)
jump: Space
crouch: Ctrl (Left)

## Combat
attack_light: Mouse Button Left
attack_heavy: Mouse Button Left (Hold)
block: Mouse Button Right
dodge: Space (in combat mode)

## Interaction
interact: E
use_item_1: 1
use_item_2: 2
use_item_3: 3
use_item_4: 4

## UI
toggle_inventory: Tab
toggle_quest_log: J
toggle_map: M
toggle_character: C
pause: Escape

## Camera
camera_zoom_in: Mouse Wheel Up
camera_zoom_out: Mouse Wheel Down
