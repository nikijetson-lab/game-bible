# Game Bible

Біблія світу. Тут живе все: регіони, раси, персонажі, квести.

Структура:
- `regions/` — географія, локації, атмосфера
- `races/` — народи, істоти, культури
- `characters/` — ключові персонажі
- `quests/` — квестові лінії
