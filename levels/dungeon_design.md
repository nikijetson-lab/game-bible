# 🏚️ Дизайн Рівнів — Підземелля та Локації

## Типи Данжонів

### 1. Затоплена Крипта (Swamp Crypt)
```
Рівень 1: Зала Води
├── Мілководдя (повільний рух)
├── Пастки: плити, що провалюються
└── Вороги: 3 утопленики
    ↓
Рівень 2: Склеп
├── 4 саркофаги (2 фальшиві)
├── Пастки: отруйний газ при відкритті
└── Бос: Слуга Моура (гнилий лицар)
    ↓
Рівень 3: Ядро
├── Фінальний бій
├── Скарб: "Сльоза Моура" (ключовий предмет)
└── Вихід активується після перемоги
```

### 2. Вежа Божевільного Алхіміка
```
Поверх 1: Лабораторія
├── Столи з реактивами (можна створити зілля)
├── Зруйновані клітки (попередження)
└── Вороги: 2 оживлених гомункули
    ↑ ліфт ↑
Поверх 2: Сховище Інгредієнтів
├── Унікальні ресурси!
├── Охоронець: Голем із плоті
└── Загадка: потрібна правильна послідовність ритуалу
    ↑ ліфт ↑
Поверх 3: Житлові Кімнати
├── Щоденник алхіміка (історія)
└── Сейф із формулою безсмертя (квестовий предмет)
```

### 3. Катакомби Тіней (Shadow Catacombs)
```
Рівень 1: Вхід
├── Постійна темрява
├── Пастка: фальшива стіна → прірва
└── Вороги: 5 скелетів-лучників (на височинах)
    ↓
Рівень 2: Зали Спогадів
├── Дзеркала показують минуле гравця (нарратив)
├── Деякі дзеркала — портали в інші кімнати
├── Пастка: двійник гравця (бій із самим собою)
└── Нагорода: +1 до будь-якої характеристики
    ↓
Рівень 3: Гніздо Мовчазних
├── Фракційний вибір: договір або бій
└── Секретний прохід до кімнати Таємних Знань
```


### Кімнати-Пастки
```markdown
1. Кімната Стиснення (стіни рухаються, треба втекти за 10 сек)
2. Кімната Отрути (постійний DPS, треба знайти вентиляцію)
3. Кімната Ілюзій (двері закриті, треба знайти справжній вихід)
4. Кімната Лузерів (з'являються копії переможених ворогів)
```

### Скарби
```gdscript
func generate_loot(room_level: int) -> Dictionary:
    var possible = {
        "common": ["50 золотих", "Зілля здоров'я", "Стріли x10"],
        "rare": ["100 золотих", "Рунічний камінь", "Зілля мани"],
        "epic": ["250 золотих", "Ключ від наступного рівня", "Частина сета"],
        "unique": ["500 золотих", "Кривавий самоцвіт", "Легендарна зброя"]
    }
    
    # Шанси залежать від складності
    var chance = randf()
    if chance < 0.05:   return possible["unique"].pick_random()
    elif chance < 0.2:  return possible["epic"].pick_random()
    elif chance < 0.5:  return possible["rare"].pick_random()
    else:               return possible["common"].pick_random()
```
