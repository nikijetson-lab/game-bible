# Крафт

## Мета

Крафт має залишатись тісно обмеженим підготовкою до справи. Це не окрема гра всередині гри — це інструмент для вирішення проблем.

## Три категорії

### 1. Польові витратні матеріали

Одноразові предмети для вирішення специфічних проблем.

Приклади:
- **Болотяна мазь:** Захист від отруйних укусів на 24 години
- **Протиотрута від пропасниці:** Лікує болотяну пропасницю
- **Приманка для Очеретяних:** Відволікає або приваблює мертво-водяні явища
- **Сигнальний вогонь:** Позначає позицію для союзників
- **Заспокійливий дим:** Заспокоює агресивних болотяних істот

### 2. Інструменти Вартівника

Багаторазові інструменти, що розширюють опції гравця.

Приклади:
- **Слідчий набір:** Інструменти для аналізу слідів, волокон, ґрунту
- **Печатка Вартівника:** Офіційний документ, що відкриває двері
- **Ліхтар Святої Вей:** Захищає від певних надприродних загроз
- **Казанок для ритуалів:** Використовується в ритуальних діях
- **Капкани:** Для полювання або захисту периметра

### 3. Спеціальна підготовка

Унікальні предмети для конкретних квестів або загроз.

Приклади:
- **Ритуальні матеріали:** Попіл, очерет, кістки — для завершення поховальних ритуалів
- **Підроблені документи:** Для обходу бюрократії
- **Спеціалізована отрута:** Для зброї проти конкретного типу ворога
- **Вибухівка:** Для руйнування перешкод

## Ресурси

Ресурси поділяються на:

- **Універсальні:** Звичайні матеріали, доступні в будь-якому регіоні (віск, шнури, металеві обрізки)
- **Регіональні:** Специфічні для Хейзмуру (болотяне дерево, торф, рідкісні трави, попіл святині)
- **Квестові:** Отримуються тільки через специфічні дії або розвідку

## Правило дизайну

Якщо предмет не можна описати одним реченням «що він робить і коли його використовувати» — він зайвий.
